# JVA Montagens

Sistema completo para gerenciamento da JVA Montagens. 

Backend: JAVA + Spring-Boot

Frontend: Javascript + NextJS

Plataforma de gestão financeira e operacional que centraliza o controle de clientes, parques, competências mensais, registros de serviços de montagem e pagamentos. O sistema oferece um dashboard executivo com indicadores financeiros em tempo real, visão de lucro total e lucro mensal, além de CRUD completo para todas as entidades com operações de criação, edição e exclusão.
